Pertanyaan dan Jawaban Seputar KirimWA.id

Apa itu KirimWA.id?

KirimWA.id adalah layanan gratis untuk mulai melakukan chat di WhatsApp tanpa harus menyimpan nomor ke dalam kontak terlebih dahulu.

Apakah saya harus install dulu KirimWA.id?

TIDAK. KirimWA.id adalah layanan berbasis web anda tidak perlu melakukan instalasi. Cukup kunjungi alamat https://kirimwa.id pada browser pada perangkat anda seperti Google Chrome atau Safari.

Apakah KirimWA.id dapat berjalan di iPhone dan Android?

YA. KirimWA.id dapat berjalan di iPhone, Android atau komputer. Pada komputer KirimWA.id otomatis akan membuka WhatsApp Web.

Bagaimana kebijakan privasi KirimWA.id?

KirimWA.id sangat menghargai privasi anda. Semua data yang tersimpan tidak akan kami bagikan kepada pihak ketiga dan tidak pula dikomersialkan. Sehingga anda dapat yakin bahwa data anda aman pada KirimWA.id.
